# CapCut Version Downgrader

This script allows installing and maintaining a specific older version of CapCut (1.4.0.198).
You can change the version to downgrade to by modifying the batch file.

## Purpose

Some users prefer older versions due to UI changes, performance differences, or removed features.

This tool:

* Installs a specific version using winget or a local backup
* Prevents multiple versions from accumulating
* Keeps a backup of the installer for offline use

## Disclaimer

This project does NOT modify CapCut, bypass licensing, or enable premium features.

It only automates installation and version management using publicly available installers.
